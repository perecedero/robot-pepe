Funciones de Pepe

pepe.mover( pasos )

Mueve el robot tantos lugares como diga "pasos". Si Pepe calcula que se sale del mapa entonces no se moverá

"pasos" es un numero entero positivo. 

	pepe.mover( 3 );
pepe.girar( direccion )

Gira a Pepe hacia el lado que diga "direccion".

"direccion" puede ser 'derecha' o 'izquierda'.

	pepe.girar( 'derecha' );

Funciones Matemáticas

es_igual( x, y )

Retorna verdadero si "x" es igual a "y", y falso en caso contrario

"x, y" deben ser números enteros. 

	es_igual( 3,  5 );
	es_igual( 3,  pepe.engranajes );
es_distinto( x, y )

Retorna verdadero si "x" es distinto a "y", y falso en caso contrario

"x, y" deben ser números enteros.

	es_distinto( 3,  5 );
	es_distinto( 4,  pepe.pos.x );
es_par( x )

Retorna verdadero si "x" es un numero par y falso en caso contrario

"x" debe ser un número entero.

	es_par( 3 );
es_impar( x )

Retorna verdadero si "x" un numero impar, y falso en caso contrario

"x" debe ser un número entero.

	es_impar( 3 );

Funciones de control de flujo

si( condicion, funcion )

Ejecuta la funcion dada en "funcion" si y solo si "condicion" es verdadera

"condicion" debe ser algo que retorne verdadero o falso.

"funcion" debe ser un bloque de código a ejecutar.

	si( es_igual(3,3), function() {
		pepe.mover(5);
		pepe.girar('derecha');
	});
repetir( x, funcion ( i ) { })

Ejecuta la función dada en "funcion" tantas veces como indica "x"

"x" debe ser un entero positivo.

"funcion" debe ser un bloque de código a ejecutar.

"i" informa la repetición actual.

	repetir( 4, function( i ) {
		//si esta es la tercera repetición i es igual a 3
		pepe.mover(5);
		pepe.girar('derecha');
	});

Recorrer un cuadrado de 5x5 ejecutar

		pepe.mover(5);
		pepe.girar('derecha');
		pepe.mover(5);
		pepe.girar('derecha');
		pepe.mover(5);
		pepe.girar('derecha');
		pepe.mover(5);
		pepe.girar('derecha');

Recorrer un cuadrado de 5x5 mejorado ejecutar

	repetir( 4, function() {
		pepe.mover(5);
		pepe.girar('derecha');
	});

Recorrer un cuadrado de 14x5 ejecutar

	repetir( 4, function( i ) {

		si( es_impar( i ), function(){
			pepe.mover(14);
			pepe.girar('derecha');
		});

		si( es_par( i ), function(){
			pepe.mover(5);
			pepe.girar('derecha');
		});

	});
